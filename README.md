# 🎓 Student Performance Analysis & GPA Prediction using Machine Learning

> A data science project analyzing how academic habits, lifestyle, and mental health factors influence student performance.

---

## 🚀 Overview

This project explores how various factors such as **study habits, sleep patterns, mental health, and digital behavior** impact student GPA.

It combines **Exploratory Data Analysis (EDA)** with a **Machine Learning model** to extract insights and predict academic performance.

---

## 🎯 Problem Statement

Student performance is influenced by multiple factors beyond academics. This project aims to:

* Identify key factors affecting student GPA
* Analyze the impact of lifestyle and mental health
* Understand the effect of digital behavior (screen time)
* Build a predictive model for academic performance

---

## 📊 Dataset

Due to size limitations, the dataset is not included in this repository.

📥 **Dataset Source:**
https://www.kaggle.com/datasets/sharmajicoder/college-students-habits-and-performance/data

### Dataset Features:

* 📚 Academic data (study hours, attendance, scores)
* 🧠 Mental health indicators (stress, focus)
* 🏃 Lifestyle factors (sleep, physical activity)
* 📱 Digital behavior (screen time, social media usage)

**Target Variable:** GPA (Grade Point Average)

---

## 🔍 Exploratory Data Analysis (EDA)

The project performs detailed analysis to uncover patterns and relationships:

* Feature distributions and correlations
* Heatmap visualization
* Pairplot analysis
* Impact analysis of:

  * Sleep hours on GPA
  * Parent education level on GPA
  * Study habits on performance

---

## 📈 Key Insights

* 📚 Study hours have a strong positive correlation with GPA
* 😴 Adequate sleep significantly improves academic performance
* 📱 Excessive screen time negatively impacts GPA
* 🧠 Mental health plays a crucial role in student outcomes

---

## 🤖 Machine Learning Model

* **Model Used:** Random Forest Regressor
* **Objective:** Predict student GPA based on multiple features

### Workflow:

* Data preprocessing
* Feature selection
* Train-test split
* Model training and evaluation
